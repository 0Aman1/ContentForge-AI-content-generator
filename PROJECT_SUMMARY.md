## 🚀 Quick Start

### 1. Install Dependencies (2 min)
```bash
pip install -r requirements.txt
```

### 2. Install AI Model (5-10 min)
```bash
ollama run llama3
```

### 3. Run the App (1 min)
```bash
streamlit run app.py
```

### 4. Open Browser
```
http://localhost:8501
```

**That's it! You're ready to generate content!**
